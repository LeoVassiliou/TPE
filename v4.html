How This Works

Rate Core Premises

Each argument has core premises. Slide to your confidence (0–100%). Core conclusion = product of all premise probabilities.

Read the Evidence Carefully

Toggle to see supporting evidence for each premise — the arguments are much stronger than popular discourse suggests. Read before rating.

Engage the Objections

Objections are listed with their rebuttals. Note whether you find the rebuttal decisive before adjusting your rating.

Evaluate Post-Argument Properties

Given the conclusion holds, how likely is each divine property? Your confidence combines with core probability.

Properties Aggregate Globally

Multiple arguments supporting the same property combine via noisy-OR: more independent evidence = higher probability.

Entailment Flows Automatically

"Christian God" entails all properties. "Maximally Great" entails most. Evidence flows without double-counting.

Math:
• Core conclusion = P₁ × P₂ × … × Pₙ
• Property contribution = core × post-premise
• Combined property = 1 − ∏(1 − contributionᵢ) (noisy-OR)
• Combo = max(∏ combinedProp(pₖ), christianGodProb) when Christian God selected

📖 Glossary
Noisy-OR
Probabilistic combination: P(effect) = 1 − ∏(1 − P(causeᵢ)). More independent evidence = higher probability. Prevents double-counting.
Entailment
If A is true, B must be true. "Christian God" entails "Personal" — if Christian God exists, He is necessarily personal.
S5 Modal Logic
If possibly necessary, then necessary (◇□P → □P). Used in the Ontological Argument to show possibility entails actuality for necessary beings.
Properly Basic
Beliefs rational without inferential support, like "other minds exist" or "the past is real." Not everything needs a proof.
PSR
Principle of Sufficient Reason: everything has an explanation for its existence — either in its nature (necessary being) or in an external cause.
BGV Theorem
Borde-Guth-Vilenkin (2003): any universe with average positive expansion must have a past spacetime boundary. Applies to virtually all cosmological models.
