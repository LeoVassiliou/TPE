How This Works

Rate Core Premises

Each argument has core premises. Slide each to your confidence level (0–100%). The core conclusion probability is the product of these confidences.

Evaluate Post-Argument Properties

Once a core conclusion is established, post-premises ask: "Given the conclusion, does the cause have property X?" Your confidence here combines with the core probability to produce a contribution to that property.

Properties Aggregate Globally

If multiple arguments support the same property (e.g., "personal"), they combine via noisy-OR aggregation: the more independent arguments support it, the higher the aggregate probability.

Entailment

Some properties logically entail others. For example, "Christian God" entails transcendent, first cause, personal, maximally great, powerful, intelligent, moral source, and ground of meaning. Evidence for an entailing property automatically flows into all entailed properties.

Build Custom Combinations

In the Combinator tab, select which properties you care about and see the probability of a being having all of them — computed across every argument's evidence (including entailments).

Math notes:
• Core conclusion = P₁ × P₂ × … × Pₙ (product of premise confidences).
• Property contribution from one argument = core × post-premise confidence.
• Entailed contribution: If property A entails property B, then A's contribution also counts as a contribution to B.
• Combined property across arguments = 1 − ∏(1 − contributionᵢ) (noisy-OR — treats arguments as approximately independent evidence sources).
• Combo = ∏ combinedProp(pₖ) for selected properties.

This is a structured reflection tool, not a formal proof system. The independence assumptions are modeling simplifications. The goal is to help you think rigorously about how your confidence in individual premises cascades into a cumulative case.
